## Project Description (Detailed - for resume bullet points)

### Core Features:
- **Animal Adoption System**: Browse and search adoptable pets with advanced filtering by type, breed, and location
- **Animal Reporting Platform**: Report stray, lost, or found animals with photo uploads and automatic location detection
- **Location-Based Services**: Real-time geolocation API integration with automatic address geocoding using Geopy
- **NGO/Shelter Management**: Dedicated dashboard for shelters to manage animals, adoption requests, and track statistics
- **Real-Time Notification System**: Automated notifications to NGOs within 50km radius when animals are reported nearby
- **Adoption Request Management**: Complete workflow for users to request adoptions and shelters to approve/reject requests
- **User Authentication & Authorization**: Role-based access control (Regular Users vs. NGO/Shelter accounts)
- **Responsive Design**: Mobile-first design with camera integration for direct photo capture on mobile devices
- **Dynamic Form Features**: Auto-fill location fields via geolocation API, breed suggestions based on animal type
- **Image Processing**: Photo upload and management with Pillow for image handling

### Technical Achievements:
- Implemented Haversine formula for distance calculations between locations
- Built scalable notification system with read/unread status tracking
- Deployed production-ready application on Render with PostgreSQL database
- Optimized deployment configuration for free-tier hosting constraints
- Integrated environment variable management for secure configuration
